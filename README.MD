# LibreRTOS

LibreRTOS是一个使用C，汇编语言编写的RTOS。

![Bitbucket open pull requests](https://img.shields.io/bitbucket/pr-raw/LibreRTOS-Community/LibreRTOS?style=for-the-badge)

## 🎉特点
- 提供外设的硬件抽象层（HAL）。
- WIFI，USB，蓝牙等协议栈。
